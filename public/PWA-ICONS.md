# PWA icons (optional)

To get the **install icon in the browser address bar** on Chrome/Edge (Windows, Mac, Android):

1. Create two PNG images from your logo:
   - **pwa-192x192.png** — 192×192 pixels  
   - **pwa-512x512.png** — 512×512 pixels  

2. Put both files in this folder (`public/`). They are already referenced in the app manifest.

You can export from your design tool or use a free converter (e.g. [CloudConvert](https://cloudconvert.com/svg-to-png)) with `logo.svg` as source.

Without these PNGs, the app is still installable via the browser menu (⋮ → Install app).
